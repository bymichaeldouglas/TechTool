Quick Functions

This tool allows you to upload a JSON file, manipulate all entries, and have the updated JSON file automatically downloaded.

Back to Predicted:

This tool allows you to move all entries from the "annotations" section back to the "predictions" section while rewriting the predicted label values with the annotated. Used to bulk add classification to a detector label job.

Flatten This tool allows you to update all entries to be under Detector 1.

Single Object Predictions: This tool allows you to remove all duplicate entries.
